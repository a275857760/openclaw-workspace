# openclaw-workspace

OpenClaw workspace backup — scraper scripts and personal agent config files.

## Structure

```
old/
├── SOUL.md           — Agent personality & behavior guidelines
├── USER.md           — User profile & preferences
├── AGENTS.md         — Workspace instructions & memory rules
├── IDENTITY.md       — Agent identity (name, emoji, vibe)
├── TOOLS.md          — Local tool notes & credentials (token redacted)
├── BOOTSTRAP.md      — First-run setup script
├── HEARTBEAT.md      — Periodic background task template
└── scripts/          — Web scraper / data collection scripts

```

> **Note:** `old/` contains the archived snapshot of personal workspace files.
